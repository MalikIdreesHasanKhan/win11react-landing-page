An almost perfect Windows 11 remake

Over the past few months, we have made progress in transforming it from a recreation with basic functionality to an almost perfect recreation.

Open-Source

Win11React is completely open-source, meaning anyone, anytime, anywhere can contribute their vision to the project, via issues or pull requests

Built with React

Win11React, as the name suggests, is built in React.js, a free and open-source front-end JavaScript library.

One of it's kind

Win11React is the only one of it's kind to boast a desktop build, dark mode, settings, file explorer, and a fully working store.
